# *AvtoTechnikShop*

### — интернет‑магазин автозапчастей и расходников для уверенного обслуживания вашего автомобиля. Подбираем подходящие позиции, следим за качеством и быстро доставляем заказы. У нас — понятный каталог, честные цены и поддержка на каждом этапе покупки.

Cделан на **Python** **Django**.
